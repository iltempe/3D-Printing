# Summary

Keychain with texts, made with openscad (parametrizable)

# How I Designed This

## Designed using openscad

It has been designed to be a parametrizable keyring, changing size, texts and font.

It contains basic openscad functions (translate, difference, cylinder, resize, text with custom fonts, hull, use of external scad files).

If you want to edit this keyring, you'll have to set a TTF font and update the first lines in the .scad file.
