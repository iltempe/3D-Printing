# Summary

I designed this vase as a pen holder for my desk. I thought it would be fun to try to design something that was functional and attractive.

The Vase is 100mm H x 85mm W.

** Print using VASE MODE. The STL is a solid form and must be printed using the vase mode in your slicer of choice. I printed this using Simplify3D.**

# Print Settings

Printer: Hatchbox Alpha
Rafts: No
Supports: No
Resolution: 0.25mm
Infill: VASE

Notes: 
I printed this using a 0.4mm nozzle but with a 1mm vase wall. The extruder just flows more filament to get the proper width.
